## متى نكتب ADR؟

- القرار يمس **سلامة البيانات المالية** أو **عزل المستأجرين** أو **الأمان**.
- التراجع عنه لاحقًا يتطلب هجرة بيانات أو إعادة كتابة وحدة.
- رفضنا بديلًا واضحًا لسبب غير بديهي — والسبب يستحق الحفظ.

**لا** نكتب ADR لاختيار مكتبة أو نمط تنسيق أو تفصيل تنفيذي قابل للتغيير بلا أثر.
