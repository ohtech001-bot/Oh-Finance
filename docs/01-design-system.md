**قاعدة اللون المالي (إلزامية):**
مبلغ **مدين / دَين** = أحمر · مبلغ **دائن / مدفوع** = أخضر · رصيد **صفر** = رمادي.
لا تُستخدم هذه الألوان لأي غرض تزييني آخر.

```
┌────────────────────────────────────────────────┬──────────────┐

├────────────────────────────────────────────────┤   Sidebar    │
│                                                │   w=260px    │
│  عنوان الصفحة (h1) + أيقونة                     │   #0B1220    │
│  فُتات الخبز: الرئيسية / الزبائن                  │   ثابت       │
│                                                │              │
│  [أزرار الإجراءات — أعلى اليسار]                 │   العناصر     │
│  [شريط الفلاتر — بطاقة بيضاء]                    │   الـ12      │
│  [بطاقات KPI — شبكة 4-5 أعمدة]                   │              │
│  [الجدول / الرسوم — بطاقة بيضاء]                  │   ────────   │
│  [الترقيم]                                       │  الدعم       │
└────────────────────────────────────────────────┴──────────────┘
        المحتوى: bg #F8FAFC · حشو 24px
```

**الاتجاه:** `dir="rtl"` على `<html>`. الشريط الجانبي على **اليمين** (وهو `inline-start` في RTL).
**إلزامي:** خصائص منطقية فقط (`ps-6`, `me-4`, `border-s`) — **ممنوع** `left`/`right` مباشرة، لأنها تكسر التبديل إلى العبرية أو الإنجليزية.

### 2.1 عناصر الشريط الجانبي (بالترتيب)

```
لوحة التحكم · الزبائن · الطلبات · الدفعات · الحسابات والحركات
التقارير · المستندات والطباعة · الرسائل والواتساب
المنتجات (اختياري) · الموظفون · الإعدادات · الاشتراك والفواتير
─────────────
الدعم والمساعدة  (مثبّت أسفل، بطاقة داكنة أفتح)
```

**العنصر النشط:** خلفية `#1D4ED8` بعرض كامل + نص أبيض + أيقونة بيضاء.
**الخمول:** نص `#94A3B8`، عند التحويم خلفية `rgba(255,255,255,.05)`.

### 2.2 نقاط التوقف

| النقطة | السلوك |
|---|---|
| `≥1280px` | شريط جانبي ثابت + جداول كاملة |
| `768–1279px` | شريط جانبي مطوي (أيقونات فقط، 72px) + جداول بتمرير أفقي |
| `<768px` | شريط سفلي + FAB + قائمة منزلقة + **بطاقات بدل الجداول** |

## 3. المكوّنات

### 3.1 بطاقة KPI

```
┌──────────────────────────────┐
│  التسمية (13px, muted)   [🔲]│  ← أيقونة 48×48، radius 12، خلفية ملوّنة 10%
│  38,450.00 ر.س  (26px, bold) │  ← ملوّن حسب الدلالة المالية
│  من 56 زبون (12px, muted)     │  ← نص فرعي أو دلتا (↗ +15%)
└──────────────────────────────┘
bg:#FFF · radius:12 · border:#E2E8F0 · shadow-card · pad:20
```
الدلتا الموجبة: `↗ +15%` بالأخضر. السالبة: `↘ -8%` بالأحمر.

### 3.2 شارات الحالة (Status Pill)

`radius: 6px · padding: 4px 10px · font: 12px/600`

| الحالة | الخلفية | النص |
|---|---|---|
| مدفوع / دائن / نشط | `#DCFCE7` | `#15803D` |
| مدفوع جزئيًا | `#FEF3C7` | `#B45309` |
| مؤكد | `#DCFCE7` | `#166534` |
| عرض سعر | `#DBEAFE` | `#1D4ED8` |
| مسودة | `#F1F5F9` | `#475569` |
| ملغي / مدين / غير نشط | `#FEE2E2` | `#DC2626` |
| لا يوجد رصيد | `#F0FDF4` | `#16A34A` |

بعضها يسبقه نقطة ملوّنة `6px` (كما في بطاقات إحصاء الطلبات).

### 3.3 الجدول

```
رأس:    bg:#F8FAFC · نص 13/600 · #475569 · ارتفاع 48px
صف:     ارتفاع 56px · فاصل سفلي #EEF2F6 · hover bg:#F8FAFC
الرقم:  عمود # أول (من اليمين)
المعرّف: رابط أزرق #1D4ED8 (CUST-0001، ORD-00087، PAY-00045)
المبالغ: محاذاة لليمين · tabular-nums · ملوّنة بالدلالة المالية
إجراءات: زر مربع 36×36 بحدود · «⋯» · آخر عمود (يسار)
```
**تذييل الجدول:** `عرض 1 - 10 من 95 زبون` (يمين) · اختيار «لكل صفحة» · ترقيم `السابق [1] 2 3 … 10 التالي` (يسار).

### 3.4 شريط الفلاتر

بطاقة بيضاء أفقية: `بحث سريع ⌕` · `من تاريخ 📅` · `إلى تاريخ 📅` · `كل الزبائن ▾` · `كل الحالات ▾` · `تصفية متقدمة ⛛`.
الارتفاع `44px` لكل عنصر · `radius: 10px` · حدود `#E2E8F0`.

### 3.5 الأزرار

| النوع | المظهر | مثال |
|---|---|---|
| أساسي (أخضر) | `bg:#15803D` نص أبيض | «إضافة طلب جديد» · «حفظ التغييرات» |
| أساسي (أزرق) | `bg:#1D4ED8` نص أبيض | «إضافة زبون جديد» · «تسجيل دفعة» |
| ثانوي | `bg:#FFF` حدود `#E2E8F0` | «تصدير» · «طباعة كشف حساب» |
| واتساب | `bg:#FFF` حدود + أيقونة خضراء | «إرسال رسالة» |
| خطر | `bg:#DC2626` | «إلغاء الطلب» (بحوار تأكيد) |

ارتفاع `44px` · `radius: 10px` · `font: 14/600` · الأيقونة على يسار النص.

### 3.6 الرسوم (Recharts)

- **Donut** (توزيع الديون / الفئات / طرق الدفع): `innerRadius 60%` · وسيلة إيضاح على اليمين مع نقاط ملوّنة ونِسَب.
- **خطي مزدوج** (المبيعات مقابل المقبوضات): أزرق `#2563EB` (مقبوضات) + أخضر `#16A34A` (مبيعات) · نقاط `r=3` · شبكة أفقية `#F1F5F9`.
- **أعمدة** (الطلبات حسب اليوم): بنفسجي فاتح `#C4B5FD` · radius علوي `4px`.
- **RTL:** المحور السيني `reversed` · محور القيم على اليمين.

## 5. الحالات (إلزامية لكل شاشة بيانات)

| الحالة | التنفيذ |
|---|---|
| **تحميل** | Skeleton يطابق شكل المحتوى (صفوف الجدول / البطاقات) — لا spinner عام |
| **فارغ** | أيقونة + عنوان + وصف + زر إجراء أساسي («لا يوجد زبائن بعد» + «إضافة زبون») |
| **خطأ** | أيقونة حمراء + رسالة + زر «إعادة المحاولة» |
| **لا نتائج فلترة** | «لا توجد نتائج مطابقة» + زر «إعادة تعيين الفلاتر» |
| **تأكيد** | حوار لكل عملية غير قابلة للتراجع (إلغاء طلب، عكس دفعة، إغلاق دورة) |
| **تغييرات غير محفوظة** | تحذير عند مغادرة نموذج مُعدَّل |
| **Toast** | نجاح (أخضر) / خطأ (أحمر) — أعلى يسار |

## 6. الوصولية

- تباين `≥ 4.5:1` للنص العادي، `≥ 3:1` للنص الكبير — **مُتحقق منه بـ axe-core في CI**.
- كل عناصر التفاعل قابلة للوصول بلوحة المفاتيح، مع حلقة تركيز مرئية `2px #1D4ED8`.
- `aria-label` على كل زر أيقوني (`⋯`, الجرس, الطباعة).
- الجداول: `<caption>` مخفي بصريًا + `scope` على رؤوس الأعمدة.
- الحوارات: حبس التركيز + `Esc` للإغلاق + `aria-modal`.
- **اللون ليس الوسيلة الوحيدة** لنقل الحالة — كل شارة تحمل نصًا صريحًا.
- `lang` و `dir` يتغيّران مع اللغة (`ar`/`he` = rtl، `en` = ltr).
